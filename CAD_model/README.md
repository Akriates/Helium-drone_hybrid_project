## CAD Models

This folder contains the different model versions developed throughout the project. These models represent multiple design iterations created from repeated testing and refinement.

- **Full Model (Version 1)**  
  This is the first and most complete design, created as a reverse-engineered approximation of an existing helium drone product. It represents the best attempt at replicating a functional system based on available observations and understanding.

- **Final Parts Configuration**  
  This version focuses primarily on the **electronics packaging and integration**, ensuring that all components could be physically mounted and operated together for a self-contained test setup.

- **Version 2 (Conceptual Design)**  
  This is a theoretical redesign developed after insights gained from building and testing the initial prototype. It reflects improvements in structure, layout, and overall system considerations.

Overall, these models document the design evolution of the project, from initial replication to more informed and refined concepts.
